# Dyslexia Reader

A modern Chrome extension designed to improve online readability and focus for students and readers with dyslexia.

---

## Features

- OpenDyslexic font support
- Adjustable text size and spacing
- Focus Mode for active paragraph reading
- Reading ruler with mouse-follow mode
- Auto-follow scroll guidance
- Read Aloud support
- Light and Sepia themes
- Smooth animations and lightweight performance

---

## Why I Built This

Many accessibility extensions feel outdated, cluttered, or difficult to use.

I wanted to create a cleaner and more modern reading assistant that genuinely improves readability while staying lightweight and simple.

This extension was built with real testing and iterative UI improvements focused on accessibility and reading comfort.

---

## Tech Stack

- JavaScript
- HTML
- CSS
- Chrome Extension APIs

---

## Installation

1. Download or clone this repository
2. Open Chrome
3. Go to:

chrome://extensions/

4. Enable Developer Mode
5. Click “Load Unpacked”
6. Select the extension folder
